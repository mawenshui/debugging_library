# 后续功能优化

本文档用于记录“尚未完成但计划做”的功能点；已落地的能力会标注为“已完成”。

## 已完成

1. 数据清理（危险）：支持按职业/标签/更新时间范围批量物理删除问题（删除前需本机操作密码校验）。
2. 批量导出：将问题与关联信息批量导出为 Excel/CSV/JSONL，支持按职业/标签/更新时间范围筛选。
3. 表单导入：支持从 Excel（.xlsx）导入问题数据，支持字段映射与校验报告。
4. 搜索增强：更智能的摘要/snippet 与高亮；更强的模糊匹配（基于 FTS5 的可配置检索策略，且可在无结果时回退到包含匹配）。
5. 局域网交换：个人端与总库通过局域网进行双向交换（支持拉取导入/推送导入；支持共享密钥鉴权）。
6. UI 一致性：DatePicker、DataGrid、PasswordBox 等控件全局主题化，避免默认样式导致的对比度与风格不一致。
7. 窗口一致性：全局自定义标题栏样式与交互（拖拽/双击最大化/右键系统菜单/最小化最大化关闭）。
8. 关于窗口：默认尺寸与布局优化，内容可完整显示；窗口较小时提供滚动条。
9. 文字清晰度：浅色背景下统一文字渲染策略与像素对齐，减少发虚与锯齿。

## 待做（建议优先级从高到低）

1. 权限与安全：在离线场景下增强权限与隐私（导出包加密/验签、脱敏策略、审计日志、口令策略与锁定策略）。
2. 导出包可信链：为离线包引入签名证书/可信公钥配置，支持校验失败阻断导入与可视化风险提示。
3. 数据备份与恢复：一键备份数据库/附件/配置；支持从备份恢复与“只读打开备份”排查问题。
4. 变更历史与可追溯：问题/标签/附件的变更历史（字段级 diff）、操作人、操作时间与来源（导入/手工编辑）。
5. 数据质量检查：缺失字段、重复问题、无效附件、异常时间戳等巡检；提供修复向导与批量修复。
6. 高级检索：语法检索（字段限定、范围过滤、NOT/OR）、同义词与拼写纠错、搜索历史与收藏条件。
7. 批量编辑：支持对命中列表批量打标签、批量修改职业/状态、批量补充环境字段（带预览与回滚点）。
8. 导入映射模板：Excel 字段映射可保存/复用；支持多模板、自动识别表头、导入前预览与“仅校验不写入”。
9. 附件管理增强：附件去重展示、缺失/损坏检测、批量导入附件、文本附件快速检索（按内容检索/高亮）。
10. 性能与大数据量体验：列表虚拟化、渐进加载；导入/导出/交换支持进度、可取消与失败后续传。
11. 角色/职业体系增强：支持自定义职业、字段模板版本化、模板导入导出；支持团队统一模板分发。
12. 多实例协作体验：局域网交换增加可视化握手/权限提示/连接诊断；支持自动发现同网段节点（可选）。
13. 可用性与可访问性：键盘快捷键、焦点顺序优化、字号缩放、对比度模式与无障碍朗读友好标签。
14. 配置与环境隔离：多工作区（不同库/不同附件目录/不同配置集）切换；支持只读模式与临时会话模式。
15. 质量与发布：自动化 UI 回归脚本、兼容性矩阵（Win10/Win11/高 DPI）、离线升级包与校验策略文档化。
