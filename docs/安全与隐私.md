# 安全与隐私（建议版）

## 当前状态

- 当前实现仅做完整性校验（sha256），未加密、未签名
- 适用于可信环境下的 U 盘/内网共享盘传输
- 已提供“本机操作密码”用于高危操作校验（例如批量物理删除），仅保存 PBKDF2 哈希，不保存明文

## 风险点

- 导出包可被任意读取（包含问题内容、环境信息、附件）
- 包可被替换（虽然会被校验发现，但不能证明“是谁导的包”）
- 操作密码仅用于本机 UI 防误操作/校验，并不对数据库或离线包提供加密保护

## 建议增强（后续优化）

- 加密：导出包 AES-GCM（口令派生密钥，或企业下发密钥）
- 签名：企业端私钥签名 manifest，个人端内置公钥验签
- 脱敏：环境字段与附件按策略脱敏/过滤
- 安全日志：导入/导出记录写入审计表并可导出
